# Crypto Portfolio
A crypto portfolio tracker with insight into crypto market data.

## Features
* User Authentication
* Portfolio (Balance, New entries, Profit and Loss)
* Marketcap Data (Volume, Ranking)

Uses the following:
- Python
- Django
- Javascript
- Tailwind
- SQLite
